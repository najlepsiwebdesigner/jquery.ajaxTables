Ajax tables

Imagine you have situation like this:

  • You have a REST server, which is able to paginate listed data and tell you count of all data
  • You want to display this data in simple html table and do the pagination automatically in client
  • You would prefer solution, which is compatible with most of front-end frameworks as twitter's bootstrap and zurb's foundation
  • You like using front-end html snippets as components and jQuery plugins
  • You want something really lightweight and easily customizable


Ajax tables are solution,
which you were looking for!
